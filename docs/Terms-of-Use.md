## Microsoft 365 Tenant Use

UHV is installed and operated inside the customer's Microsoft 365 tenant. Tenant administrators and site owners are responsible for deciding where the app is installed, which SharePoint pages use it, and which HTML content it renders.

## Hosted Content

UHV renders HTML content selected by tenant administrators or site owners. Evotec Services sp. z o.o. does not control customer-hosted HTML files, embedded third-party content, or external resources configured by the customer.

Administrators should use UHV security settings to limit sources to trusted SharePoint paths or trusted hosts.
