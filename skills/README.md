# UHV Project Skills

Reusable operator/contributor playbooks for common UHV workflows.

## Available Skills

- `skills/uhv-release-deploy/SKILL.md`
  - Build/package, deploy to app catalog, update sites, verify installed version.
- `skills/uhv-sharepoint-debug/SKILL.md`
  - Triage SharePoint runtime issues: download behavior, deep-link failures, scroll jumps, and iframe quirks.
- `skills/uhv-tenant-profile-setup/SKILL.md`
  - Create local auth/tenant profile and validate connectivity quickly.

## Usage Pattern

1. Open the relevant `SKILL.md`.
2. Run the steps in order.
3. Capture outputs in PR notes when making operational changes.
